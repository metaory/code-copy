Code Copy

Click code to copy. Hold Alt and click other text.

Turn on for this tab

Enable Code Copy on a tab before you copy anything there.

  • Gray icon: off on this tab.
  • Color icon: on on this tab.
  • Click the toolbar icon or press Alt+C to toggle.
  • The icon matches the tab you are on. Other tabs keep their own state.
  • After navigation, enable again on the new page.

Default toggle: Alt+C. Remap at chrome://extensions/shortcuts if it clashes.

Copy code

Click a <code> block or a <pre> without nested code inside.

You get a Copied toast and a short highlight on the block.

Copy any text with Alt

Hold Alt. The copy cursor appears and text under the mouse gets an outline.

Click while Alt stays down to copy that element's visible text.

Alt pick is not the Alt+C toggle.

No popup. No options page.
