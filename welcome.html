Code Copy

Click code to copy.

Hold Alt and click any element to copy its text.

Toolbar

  • On: full-color icon; click to turn off.
  • Off: grayscale icon; click to turn on.
  • State lasts for this browser session and resets when you quit Chrome.

Code blocks

  • Click a <code> element, or a <pre> with no nested code.
  • A Copied toast and brief highlight confirm success.

Any element

  • Hold Alt — copy cursor; elements with text are outlined.
  • Click while Alt is held — copies that element’s visible text.

No popup or options page. only the toolbar toggle. Visit any page and try it.
